# Sample brand brief

- Product: a calm professional AI workbench
- Background: `#F5F3EE`
- Panel: `#FAF9F6`
- Text: `#1D1B16`
- Accent: `#DA7756`
- Secondary: `#1B365D`
- UI font: a locally available serif or system fallback
- Code font: `SF Mono`, then a monospace fallback
- Visual direction: warm paper, restrained editorial hierarchy, low-noise geometry
- Required states: homepage, task route, selected session, suggestion cards, composer, top tabs
- Image request: optional ultra-wide homepage illustration with quiet left-side text space
- Recovery requirement: preserve the current native theme export and provide an exact restore command
