# 本地资料智能分类规则 + 客户收集清单

## 智能分类规则

### 分两步走

**第一步：文件名关键词匹配**

| 文件名包含关键词 | 归入目录 |
|---|---|
| 公司简介、企业介绍、about、团队、文化、资质、荣誉、营业执照 | 企业介绍/ |
| 产品、手册、参数、规格、目录、报价、price、spec、catalog | 产品资料/ |
| 客户、案例、合作、项目、评价、testimonial、case | 客户案例/ |
| 公众号、文章、新闻、宣传册、白皮书、讲话、发言、采访 | 对外发声/ |
| 话术、销售、客服、FAQ、常见问题、投标、标书、SOP | 销售服务/ |
| 行业、报告、标准、竞品、调研、市场、GB、ISO | 行业知识/ |
| Logo、VI、品牌、视觉、色值、字体 | 品牌视觉/ |

**第二步：文件内容判断（关键词未命中时）**

读取文件前500字（可读格式），按内容主题判断：
- 讲公司历史/团队/文化的 → 企业介绍/
- 列产品型号/参数/用途的 → 产品资料/
- 描述客户合作/应用场景的 → 客户案例/
- 有明确发文日期和文章结构的 → 对外发声/
- 有销售/服务流程的 → 销售服务/
- 有行业数据/标准引用的 → 行业知识/
- 图片类文件按文件名推断，推不出归未分类/

**无法确定的**：归入 `未分类/`，在盘点报告中列出，提示用户手动归类。

### 特殊格式处理

| 格式 | 处理方式 |
|---|---|
| PDF | 读取前500字（用文本提取） |
| Word (.doc/.docx) | 读取前500字 |
| Excel (.xls/.xlsx) | 读取sheet名和前几行，按内容判断 |
| PPT (.ppt/.pptx) | 读取标题页和前几张slide标题 |
| 图片 (.jpg/.png等) | 按文件名推断，推不出归未分类/ |
| Markdown/.txt | 读取前500字 |
| 其他格式 | 归未分类/ |

## 客户资料收集清单

以下清单可导出发给客户。客户按清单整理资料文件夹后，交付给建库团队。

---

### 企业AI分身 · 资料收集清单

为了让AI准确模拟贵企业的声音和专业知识，请提供以下资料。**有什么给什么，没有的不用编。**

#### 一、企业介绍（高优先级）

我们需要：
- 公司简介（任何版本都行，长短不限）
- 发展历程、重要里程碑
- 团队介绍、组织架构
- 企业文化、使命愿景
- 资质证书、荣誉奖项

举例：公司官网的"关于我们"页面内容、宣传册里的公司介绍、营业执照照片。

#### 二、产品资料（高优先级）

我们需要：
- 产品手册、产品目录
- 技术参数表、规格书
- 报价单、价格表
- 服务说明、售后政策

举例：发给客户的产品介绍PDF、展会上用的产品手册、技术部门出的参数表。

#### 三、客户与案例（高优先级）

我们需要：
- 客户名单（可以脱敏）
- 合作案例、项目成果
- 客户评价、感谢信
- 典型应用场景描述

举例：投标用的业绩案例、公众号发过的客户故事、客户送的锦旗照片。

#### 四、对外发声（最高优先级）

我们需要：
- 公众号历史文章（越多越好）
- 官网新闻、公司动态
- 宣传册、折页
- 领导讲话稿、采访记录
- 白皮书、行业报告

**这类资料最重要**——它决定了AI写出来的文章像不像你们自己人写的。

举例：公众号后台导出的文章、官网新闻页内容、年会领导讲话稿。

#### 五、销售与服务（中优先级）

我们需要：
- 销售话术、开场白
- 客服话术、常见问题解答
- 投标文件模板
- 服务流程SOP

举例：新员工培训用的销售话术、客服部门的FAQ文档。

#### 六、行业知识（中优先级）

我们需要：
- 行业报告、市场分析
- 技术标准、规范文件
- 竞品分析、对比资料

举例：行业协会发的报告、技术部门留存的标准文件。

#### 七、品牌视觉（低优先级）

我们需要：
- 企业Logo（高清）
- VI视觉规范（如有）
- 标准色、标准字体

举例：设计公司交付的VI手册、Logo源文件。

---

### 整理方式

1. 建一个文件夹，命名为`{企业名}-资料`
2. 在里面按上面7类建子文件夹
3. 把对应的文件放进去
4. 没有的类别不用建空文件夹
5. 整理完打包发给智见AI团队

**不需要完美分类。我们收到后会用AI自动扫描和归类。您只需要确保资料放进了对应的文件夹，文件名能看出大概内容就行。**
