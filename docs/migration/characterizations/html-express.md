# html-express characterization

- **Trigger contract:** Convert information-dense content into one structured, self-contained HTML report.
- **Boundary:** Formal PDF/PPT/landing-page work routes elsewhere; deployment is outside this Skill.
- **Workflow invariant:** Select a skeleton and reusable components, then produce a single local HTML artifact.
- **Output invariant:** Self-contained, readable HTML with no build step.
- **Resource graph:** `assets/skeleton.html`, `assets/tokens.css`, reusable component files, and the component demo.
