# enterprise-clone-builder characterization

- **Trigger contract:** Build a standardized enterprise digital-clone repository from an enterprise name, local material, and public information.
- **Boundary:** Does not write with an existing clone and does not build personal creator clones.
- **Workflow invariant:** Intake, research, extraction, voice analysis, and repository generation remain separate documented steps.
- **Output invariant:** A complete repository consumable by `enterprise-clone-writer`.
- **Resource graph:** `references/directory-spec.md`, extraction, intake, voice, and web-clipper guides.
