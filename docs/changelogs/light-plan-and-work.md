# light-plan-and-work Changelog

## 1.0.0 — 2026-07-23

- Add a manual, lightweight plan-and-execute route for bounded knowledge and engineering work.
- Keep routine plans to 3–7 observable steps and avoid durable plan files by default.
- Add explicit direct, specialist, discovery, and heavy-work routing boundaries.
- Escalate destructive, security-sensitive, migration, cross-system, multi-owner, and release work.
- Add trigger and baseline-vs-Skill output evaluation fixtures.
- Add OpenAI and cross-Harness interface metadata plus deterministic contract tests.
